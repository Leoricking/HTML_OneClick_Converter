# HTML 一鍵轉圖片／純文字 v1.2.0

適用於 Chromium 架構瀏覽器：

- Google Chrome
- Microsoft Edge
- Opera

## v1.2.0 新增功能

1. **圖片格式可選 PNG / JPG**
   - 目前畫面截圖支援 PNG 與 JPG。
   - 完整網頁長圖也支援 PNG 與 JPG。
   - JPG 會以較小檔案體積輸出。
2. **新增「複製到 OneNote（20pt）」**
   - 將主要文字內容複製成可貼到 OneNote 的格式。
   - 貼到 OneNote 後會保留 **20pt 字體大小**。
   - 若瀏覽器不支援 HTML 剪貼簿，會自動退回一般純文字複製。


## v1.2.0 新增功能

1. **另存目前網頁 PDF**
   - 一鍵開啟目前網頁的列印視窗。
   - 在「目的地」選擇「另存為 PDF」即可輸出。
   - 可使用列印視窗調整紙張方向、邊界、縮放與背景圖形。
   - 受 Chromium 安全限制，擴充功能不能略過列印視窗直接靜默儲存 PDF。

## 主要功能

1. **另存目前網頁 PDF**
   - 開啟瀏覽器列印視窗並選擇「另存為 PDF」。
2. **擷取目前畫面 PNG / JPG**
   - 儲存目前分頁可見範圍。
3. **擷取完整網頁 PNG / JPG**
   - 自動捲動頁面。
   - 分段截圖後拼接成完整長圖。
   - 完成後還原原本捲動位置。
   - 第二段開始隱藏 fixed / sticky 浮動元件，降低導覽列重複出現。
4. **下載主要內容 TXT**
   - 優先尋找 `article`、`main`、`role=main` 等主要內容區域。
5. **下載全部文字 TXT**
   - 使用頁面 `body.innerText`，保留所有可見文字。
6. **複製主要內容**
   - 可直接貼到 ChatGPT、記事本、Word 等。
7. **複製到 OneNote（20pt）**
   - 以 HTML + 純文字雙格式寫入剪貼簿。
8. **TXT 中繼資料**
   - 可選擇是否加入標題、網址與擷取時間。
9. **安全檔名**
   - 自動移除 Windows 不允許的檔名字元。
10. **自動分類**
   - 圖片下載至 `HTML轉圖片`。
   - 文字下載至 `HTML轉文字`。

## 安裝

### Chrome

1. 開啟 `chrome://extensions/`
2. 開啟右上角「開發人員模式」。
3. 按「載入未封裝項目」。
4. 選擇本專案資料夾 `HTML_OneClick_Converter_v1.2.0`。
5. 將擴充功能固定到工具列。

### Edge

1. 開啟 `edge://extensions/`
2. 開啟「開發人員模式」。
3. 按「載入解壓縮」。
4. 選擇本專案資料夾。

### Opera

1. 開啟 `opera://extensions/`
2. 開啟「開發人員模式」。
3. 按「載入未封裝的擴充功能」。
4. 選擇本專案資料夾。

## 使用

1. 開啟要轉換的網頁或本機 HTML。
2. 點工具列上的「HTML 一鍵轉換器」圖示。
3. 在「圖片格式」選擇 PNG 或 JPG。
4. 依需求選擇：
   - 另存目前網頁 PDF
   - 擷取目前畫面
   - 擷取完整網頁
   - 下載主要內容 TXT
   - 下載全部文字 TXT
   - 複製主要內容
   - 複製到 OneNote（20pt）
5. 檔案會直接進入瀏覽器預設下載資料夾。

## 本機 HTML 注意事項

若要處理 `file:///...` 本機 HTML：

1. 在擴充功能管理頁找到本擴充功能。
2. 開啟「允許存取檔案網址」。
3. 重新開啟本機 HTML 後再操作。

## 已知限制

- `chrome://`、`edge://`、`opera://`、擴充功能商店等受保護頁面不能擷取純文字。
- 可見畫面截圖通常仍可用於部分受保護頁面，但不同瀏覽器政策可能不同。
- 超長或超寬頁面可能超過 Canvas 尺寸／記憶體限制。
- 動態載入頁面建議先捲到最底，讓內容載入完成，再執行整頁截圖。
- 影片、WebGL、跨來源 iframe、持續動畫或虛擬清單可能無法完整拼接。
- 整頁截圖期間請不要切換分頁或關閉彈出視窗。
- 瀏覽器頁面縮放會影響輸出解析度；超長頁面可先調低縮放比例。
- OneNote 20pt 貼上效果依 OneNote 版本而略有差異，但支援 HTML 貼上的版本通常可保留字級。

## 權限說明

- `activeTab`：只在使用者點擊擴充功能後存取目前分頁。
- `scripting`：擷取頁面文字、取得尺寸及控制捲動。
- `downloads`：將 PNG / JPG / PDF / TXT 寫入下載資料夾。
- `storage`：記住「TXT 加入中繼資料」與圖片格式選項。

本版本不會上傳網頁內容，也不需要遠端伺服器。
